## 2. シェル

### 2.1 シェルの基礎知識
- シェルとは何か（ユーザーとカーネルの橋渡し）
- シェルの種類（bash、zsh、sh、fish）
- ログインシェルと対話型シェル
- プロンプトの見方（ユーザー名、ホスト名、カレントディレクトリ）

### 2.2 シェルの基本操作
- コマンドの入力と実行の仕組み
- コマンドの書式（コマンド名・オプション・引数）
- コマンド履歴の活用（↑↓キー、`history`コマンド）
- タブ補完（Tab キーによるファイル名・コマンド名の補完）
- キーボードショートカット（Ctrl+C、Ctrl+D、Ctrl+L、Ctrl+Aなど）

### 2.3 ファイルシステムとパス
- Linuxのディレクトリ構造（/、/home、/etc、/var、/tmpなど）
- 絶対パスと相対パス
- ホームディレクトリと `~`（チルダ）
- カレントディレクトリと `.`（ドット）、`..`（ドットドット）

### 2.4 ファイルのパーミッション
- 所有者・グループ・その他の概念
- 読み取り（r）・書き込み（w）・実行（x）権限
- `ls -l` の出力の読み方
- `chmod` による権限の変更（数値表記とシンボリック表記）
- `chown` と `chgrp` による所有者の変更

### 2.5 標準入出力とリダイレクト
- 標準入力（stdin）・標準出力（stdout）・標準エラー出力（stderr）
- 出力リダイレクト（`>`、`>>`）
- 入力リダイレクト（`<`）
- エラー出力のリダイレクト（`2>`、`2>>`）
- 標準出力と標準エラーの統合（`2>&1`、`&>`）

### 2.6 パイプとフィルタ
- パイプ（`|`）の仕組みと使い方
- パイプでよく使うコマンドの組み合わせ例
- `tee` コマンド（出力を分岐する）

### 2.7 ワイルドカードとグロブ
- `*`（任意の文字列）
- `?`（任意の1文字）
- `[]`（文字クラス）
- `{}`（ブレース展開）

### 2.8 環境変数とシェル変数
- 変数の定義と参照（`=`、`$`）
- 環境変数とシェル変数の違い
- よく使う環境変数（`PATH`、`HOME`、`USER`、`LANG`）
- `export` コマンド
- `env` と `printenv` で環境変数を確認する

### 2.9 エイリアスとシェルの設定ファイル
- `alias` コマンドでショートカットを作る
- `.bashrc` / `.bash_profile` / `.zshrc` の役割
- 設定ファイルの編集と反映（`source`）
- プロンプトのカスタマイズ（`PS1`）

### 2.10 シェルスクリプト入門
- シェルスクリプトとは
- スクリプトの作成と実行（シバン `#!/bin/bash`）
- 変数と引数（`$1`、`$2`、`$#`、`$@`）
- 条件分岐（`if`、`test`、`[ ]`）
- ループ処理（`for`、`while`）
- 関数の定義と呼び出し

## 3. コマンドリファレンス

### 3.1 ディレクトリ操作
- `pwd` - カレントディレクトリの表示
- `cd` - ディレクトリの移動
- `ls` - ファイル・ディレクトリの一覧表示
- `mkdir` - ディレクトリの作成
- `rmdir` - 空ディレクトリの削除

### 3.2 ファイル操作
- `touch` - 空ファイルの作成 / タイムスタンプの更新
- `cp` - ファイル・ディレクトリのコピー
- `mv` - ファイルの移動・名前の変更
- `rm` - ファイル・ディレクトリの削除
- `ln` - リンクの作成（ハードリンク・シンボリックリンク）

### 3.3 ファイルの内容を見る
- `cat` - ファイルの内容を表示
- `less` / `more` - ページ単位で表示
- `head` - ファイルの先頭を表示
- `tail` - ファイルの末尾を表示（`-f` オプションでリアルタイム監視）
- `wc` - 行数・単語数・バイト数のカウント

### 3.4 検索・フィルタ
- `find` - ファイルを条件で検索する
- `grep` - テキストをパターンで検索する
- `sort` - 行を並べ替える
- `uniq` - 重複行を処理する
- `cut` - 列（フィールド）を切り出す
- `tr` - 文字を変換・削除する
- `awk` - パターン処理（基本的な使い方）
- `sed` - ストリームエディタ（基本的な使い方）

### 3.5 圧縮・アーカイブ
- `tar` - アーカイブの作成と展開
- `gzip` / `gunzip` - gzip形式の圧縮・展開
- `zip` / `unzip` - zip形式の圧縮・展開
- よく使う組み合わせ（`tar czf`、`tar xzf`）

### 3.6 ユーザーとグループ管理
- `whoami` - 現在のユーザー名を表示
- `id` - ユーザーID・グループIDの表示
- `su` - ユーザーの切り替え
- `sudo` - 管理者権限でコマンドを実行
- `useradd` / `userdel` - ユーザーの追加・削除
- `passwd` - パスワードの変更
- `groups` - 所属グループの確認

### 3.7 プロセス管理
- `ps` - プロセスの一覧表示
- `top` / `htop` - リアルタイムでプロセスを監視
- `kill` / `killall` - プロセスの終了
- `jobs`、`bg`、`fg` - ジョブ制御
- `&` によるバックグラウンド実行
- `nohup` - ログアウト後もプロセスを継続

### 3.8 ネットワーク基礎コマンド
- `ping` - 通信確認
- `curl` / `wget` - URLからデータを取得
- `ip` / `ifconfig` - ネットワークインターフェースの確認
- `ss` / `netstat` - ネットワーク接続の確認
- `ssh` - リモートサーバーへの接続
- `scp` - リモートとのファイル転送

### 3.9 システム情報の確認
- `uname` - システム情報の表示
- `df` - ディスク使用量の確認
- `du` - ディレクトリのサイズ確認
- `free` - メモリ使用量の確認
- `uptime` - 稼働時間の確認
- `date` - 日時の表示

### 3.10 ヘルプの引き方
- `man` - マニュアルページの使い方
- `--help` オプション
- `info` コマンド
- `type` / `which` - コマンドの場所を調べる

## 4. エディタ

### 4.1 Linuxにおけるテキストエディタの役割
- なぜテキストエディタが重要なのか
- CUI（コマンドライン）エディタとGUIエディタ
- 代表的なエディタの紹介（Vi/Vim、nano、Emacs）

### 4.2 nano - はじめてのCUIエディタ
- nanoとは（初心者向けの使いやすいエディタ）
- ファイルを開く・新規作成する
- 基本操作（文字入力、カーソル移動、削除）
- 保存と終了（Ctrl+O、Ctrl+X）
- 検索と置換（Ctrl+W、Ctrl+\）
- コピー・カット・ペースト
- 画面下部のショートカットガイドの読み方

### 4.3 Vi/Vim - Linuxの定番エディタ
#### 4.3.1 Vimの基本概念
- VimとViの関係
- なぜVimを学ぶべきか（ほぼ全てのLinux環境に入っている）
- モードという考え方（ノーマル、インサート、ビジュアル、コマンドライン）

#### 4.3.2 最初の一歩
- Vimの起動と終了（`:q`、`:q!`、`:wq`、`ZZ`）
- ノーマルモードとインサートモードの切り替え（`i`、`a`、`o`、`Esc`）
- まずはこれだけ覚えよう（最小限の操作セット）

#### 4.3.3 カーソル移動
- 基本移動（`h`、`j`、`k`、`l`）
- 単語単位の移動（`w`、`b`、`e`）
- 行頭・行末への移動（`0`、`^`、`$`）
- ファイルの先頭・末尾への移動（`gg`、`G`）
- 指定行へのジャンプ（`:行番号`、`行番号G`）

#### 4.3.4 編集操作
- 削除（`x`、`dd`、`dw`）
- コピーとペースト（`yy`、`p`、`P`）
- 元に戻す・やり直す（`u`、`Ctrl+r`）
- テキストの置換（`r`、`R`）

#### 4.3.5 検索と置換
- 文字列の検索（`/`、`?`、`n`、`N`）
- 文字列の置換（`:s/old/new/`、`:%s/old/new/g`）

#### 4.3.6 ビジュアルモード
- 文字単位の選択（`v`）
- 行単位の選択（`V`）
- 矩形選択（`Ctrl+v`）
- 選択範囲に対する操作（削除、コピー、インデント）

#### 4.3.7 便利な機能
- 複数ファイルの編集（`:e`、`:bn`、`:bp`）
- 画面分割（`:split`、`:vsplit`）
- `.vimrc` の基本設定（行番号表示、シンタックスハイライト、インデント）
- `vimtutor` で練習しよう
